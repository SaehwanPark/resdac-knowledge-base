For most files, a table of variables will display below, with links to the variable definitions. If no table displays, or if you need additional information for a specific variable, please see the CCW website. Data documentation is available for all files and codebooks for many.
If you are unable to locate the information you need, please contact ResDAC.

|---|---|
| BENE_ID | Encrypted CCW Beneficiary ID | 
| PRACTICE_ID | Model-Assigned Practice ID | 
| PRACTICE_NAME | Name of Practice | 
| TIN | Practice's Taxpayer Identification Number | 
| UNIQUE_PATIENT_ID | Unique Model-Assigned Patient ID | 
| PATIENT_DOB | Patient's Date of Birth | 
| PATIENT_SEX | Patient's Sex | 
| PATIENT_RACE | Patient's Race | 
| PATIENT_ETHNICITY | Patient's Ethnicity | 
| IS_MEDICARE_PARTB | Identify if the Patient is Enrolled in Medicare Part B | 
| INSERT_DATE | Date the Record was Created | 
| UPDATE_DATE | Date the Record was Modified |
